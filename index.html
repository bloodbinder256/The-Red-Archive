      Memory Restoration Path

      This guide is written like a field wiki: it hints at where each Red Archive memory belongs without revealing the private code. The actual sequences must still be recovered inside the mod.

      Archive Voice

      The archive is silent.

      Overview

      Each memory group ties together related systems: dimensions, rituals, weapons, enemies, creative-only structure seals, armor, and relics. When an in-game discovery reveals a Red Archive code, enter it on the Archive Terminal tab to restore the matching record.

      Note: The guide avoids showing codes on purpose. It is meant to help players hunt, not skip the hunt.

      How to Use This Guide

      Use the guide as a spoiler-light checklist while playing. When you find a suspicious fragment, item tooltip, boss drop, structure clue, or ritual message, enter the recovered sequence on the Archive Terminal. The guide will update its restored count automatically.

      Step 1 Explore the relevant in-game system: sky travel, altar rites, choir weapons, Heaven enemies, authority bosses, sealed structures, armor sets, or relic tools.
      Step 2 Record partial clues, coordinates, and item names in Player Notes. The website saves notes locally in your browser.
      Step 3 Return to the Archive Terminal when the mod gives you a complete private sequence. Spaces, hyphens, and lowercase do not matter.

      Survival Notes

      Not every memory should be hunted the same way. Some are early exploration records, some are combat records, and some are structure-lore records. Seal blocks are a special case: they are creative-only blocks used as visual and mechanical language inside special structures, not normal craftable progression blocks.

      Seal clarification: If the guide mentions seals, investigate the structure, room, altar, or ruin that uses them. Do not expect seal blocks to appear as a standard survival item path.

      Crafting Recipe Index

      This section is the practical recipe and machine reference for DemonTime: Season Two. It favors verified behavior from the current source files over old planning notes. Use in-game JEI/tooltips as the final authority if a patch changes an ingredient or station.

      Accuracy pass: the Purifier is a four-part machine loop: input item, purifier core, water, and fuel. The Heavenly Forge uses three input slots, a fuel slot, and an output slot.

      Verified stations and interfaces

      Station / interface What it does How to think about it
      Heavenly Forge Three input slots + fuel + output. Makes forge-only materials, tools, weapons, and Fallen Worlds armor pieces. The forge is the anti-bypass station: if an item is too divine, too ruined, or too dangerous for a normal table, check here.
      Purifier Input item + purifier core + water + fuel. Converts remnants, grace items, void fragments, echo relics, and cleansing materials. The inserted core decides what “purified” means. Angelic, Abyssal, Celestial, Blood, Verdant, Infernal, and Echo cores all push different recipes.
      Heavenly Altar Prepared ritual block with candles, goblet, athame, offerings, favor costs, cooldowns, and possible backlash. The altar is a negotiation. Stabilizers reduce risk, but failed rites still matter.
      Upgrade Table Base item + upgrade material + upgrade catalyst → upgraded item with saved stat bonuses. Use this after you have gear worth investing in. Catalysts decide how hard the table pushes the item.
      Heavenly Cache Box Cloud-sealed reward container. Locked caches require a Heavenly Cache Key to open. A cache is not just a chest. It is a reward gate for Heaven-style loot and structure rewards.
      Heaven Book / Contracts Quest, favor, and contract interface. Supports bring-item objectives, favor checks, rewards, and per-player contract state. Use it as the player-facing progression ledger for Heaven work.

      How the Purifier works

      The Purifier is not a normal crafting table. It processes one input item using four requirements: an input item, a purifier core, stored water, and fuel. It can run with no core or a Basic Purifier Core for basic cleansing, but advanced outputs depend on the inserted core.

      Slot / resource Accepts Important behavior
      InputThe item being purified or transformed.Water items, fuel, and cores cannot be placed here.
      WaterWater Bucket or Water Potion.Water Bucket adds 1000 water; Water Potion adds 250 water. Max tank: 4000.
      FuelCoal, Charcoal, Blaze Rod, Lava Bucket, or items in the DemonTime fuel tag.Coal/Charcoal/tag fuel burn 1600 ticks, Blaze Rod burns 2400 ticks, Lava Bucket burns 20000 ticks.
      CoreBasic, Angelic, Abyssal, Celestial, Blood, Verdant, Infernal, or Echo Purifier Core.The inserted core decides which transformations are available.
      OutputPurified result and empty containers.Outputs only; players cannot insert items directly into this slot.

      Items, Blocks, and Systems Explained

      DemonTime: Season Two progression is easier to understand if you sort everything into four categories: items you carry, blocks you interact with, systems that change the rules, and structure-only storytelling pieces.

      Items

      Items are portable proof of progression: feathers, remnants, cores, weapons, armor, relics, and Red Archive ciphers.

      • Materials: Angelic Feathers, Gilded Feathers, Aerolite, Astrelium, Star Metal, Grace, and Titan Remnants.
      • Weapons: Heaven’s Wrath, Choir Bow, Blackhole Staff line, Aerolite tools, and Astrelium tools.
      • Relics: halos, charms, rings, pendants, belts, cache keys, remnant variants, purifier cores, goblet, and athame.
      • Archive ciphers: in-game clues that point back to this site without revealing lore for free.

      Blocks

      Blocks are either survival progression stations, building materials, reward containers, or structure-only storytelling pieces.

      • Stations: Heavenly Forge, Purifier, Heavenly Altar, Upgrade Table, and cache-related blocks.
      • World blocks: cloud matter, Heaven wood, charred wood, ash-covered ground, marble/slate ruin materials.
      • Cache blocks: locked Heaven reward containers tied to Heavenly Cache Key progression.
      • Seal blocks: creative-only structure blocks used for special ruins and containment scenes, not normal crafting.

      Systems

      Systems are the loops that make the mod feel like a season instead of a pile of items.

      • Red Archive loop: find code in-game, enter it here, restore memory, keep investigating.
      • Heaven progression: sky threshold, Heaven biome, angelic drops, divine stations, stronger gear.
      • Ritual progression: altar preparation, favor, offerings, candles, goblet, athame, success, backlash, stabilizers, and cache clues.
      • Combat escalation: Choir Bow resonance, seraphic enemies, Fallen Heaven scars, Cosmic Demon gravity pressure.
      • Structure investigation: ruins, broken churches, small altars, sealed rooms, and containment marks.

      Upgrade Table system

      The Upgrade Table is a stat-upgrade station, not a recipe output table. Put one supported item in Base, upgrade material in Material, and an Upgrade Catalyst in Catalyst. The result preserves a level, power, material family, catalyst type, and stat line on the item.

      PartBehaviorExamples
      Supported basesDamageable gear, armor, tools, weapons, bows, rings, charms, sigils, amulets, pendants, belts, and relics.Choir Bow, Heaven’s Wrath, armor, belts, charms, staffs.
      Upgrade materialsMaterial family and tier decide the stat direction and cost. Cost rises with upgrade level and caps at 12 materials.Aerolite = speed/utility, Astrelium = star power, Titan/Netherite = defense, Celestial = health/refinement.
      CatalystsCatalyst profile changes strength, cost, max level, and stat bias.Lesser, Standard, Focused, Aerolite, Celestial, Titan, Volatile, Empowered.
      Level capThe global max is +8, but weaker catalysts may stop earlier.Lesser stops around +4; standard reaches +6; stronger catalysts can reach +8.

      Worldgen, Structures, and Special Blocks

      The source pack confirms the wiki should separate survival progression from structure language. Fallen Heavens ruins, broken churches, charred houses, fountains, path remnants, small altars, forge structures, star clusters, and the Cosmic Demon arena are exploration targets. Seal blocks and markers are not normal survival recipes.

      Fallen HeavensScar biome with ash, charred wood, ruin networks, and containment-style storytelling.
      Heaven BiomePale Heaven environment connected to the sky-threshold route and angelic progression.
      Cosmic Demon ArenaBoss structure tied to gravity relics, black-hole pressure, and major escalation.
      Sealed structuresHandcrafted/special builds that use seal markers and containment blocks as visual/mechanical language.
      Heavenly Cache BoxLocked reward container. A Heavenly Cache Key is required, then dissolves when used.
